# User Management Dashboard

A responsive healthcare user management dashboard built with React and Vite.

## Features
- User management
- Family member management
- Order history
- Payment tracking
- Prime user system
- Responsive admin dashboard UI

## Tech Stack
- React.js
- Vite
- JavaScript
- CSS

## Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/shubhamshubh1105/user-management-dashboard.git

Navigate to project folder
cd user-management-dashboard

Install dependencies
npm install

Run development server
npm run dev
